Application Overview

FitFocus is a fitness-tracking web application where users can log their workouts, set fitness goals, and view their progress.

Technology Overview

FitFocus is built using the following technology:

  • HTML/CSS for structure, styling
  • JavaScript for dynamic behavior
  • React for front-end development
  • Go for backend with GraphQL API
  • Chart.js for data visualization
  • Microsoft Azure for hosting

Proposal Last Updated

November 3, 2024

Feature List

  • User authentication - secure profile, sign up and log in
  • Workout logging functions
  • Goal setting
  • Progress visualization with chart
  • Personalized helpful fitness comments based on user data

Future Feature List

  • Social features for sharing progress and collaboration
  • Custom workout plans for each individual user

  • Basic user authentication - sign up and log in
